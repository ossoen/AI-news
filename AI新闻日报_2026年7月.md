## 📅 2026年7月18日

### 1. OpenAI 开源 Codex CLI，编程助手进入终端时代

OpenAI 正式将 Codex CLI 在 GitHub 上开源。这是 OpenAI 首个面向终端开发场景的开源项目，支持自然语言驱动的自主编程、命令执行和文件操作。与传统的 IDE 插件不同，Codex CLI 更像一个"坐在你旁边帮你写代码的工程师"。

随着 Codex CLI、Cursor Agent 等新一代自主编程工具的崛起，编程工具的竞争从"补全速度"转向了"理解能力和自主执行能力"。

## 📅 2026年7月12日

### 1. GPT-5.6 Sol Ultra 用 64 个子智能体 1 小时内完成 50 年图论猜想证明

OpenAI 宣布新发布的 **GPT-5.6 Sol Ultra** 生成对「圈双覆盖猜想」（Cycle Double Cover Conjecture）的完整证明，该问题在图论中悬而未决约 50 年。模型动用 64 个并行子智能体，耗时不到 1 小时，论文与提示词以 PDF 公开。

曼彻斯特大学数学家 Thomas Bloom 称证明"非常漂亮、初等"，但批评论文未引用 1983 年 Bermond 等人的先行工作。Bloom 同时指出，真正可复用的是 OpenAI 公开的提示工程——要求模型至少计算 8 小时才可放弃，而模型 1 小时即完成。

**来源**: [Swil-News](https://news.supwil.com/news/ai-tech/2026-07-12)
